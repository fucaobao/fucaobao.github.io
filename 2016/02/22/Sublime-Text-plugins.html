Sublime Text 常用插件

  • 作者 fucaobao
  • 最后编辑 2016年02月22日
  • 标签 Sublime

Sublime Text是前端的一个神器，下面是我工作这么长时间以来总结出的比较好用的插件。

  1. Alignment

    变量自动对齐

  2. Bracket Highlighter

    括号自动匹配

  3. Color Highlighter

    CSS颜色高亮

  4. Doc Blockr

    优美的注释

  5. Javascript Beautify

    JS代码格式化，更加优美

  6. Minifier

    依赖NodeJS，压缩JS文件

  7. NodeJS

    用于支持gulp、JSHint等工具

  8. SideBar

    左侧菜单栏增强工具

  9. SublimeTmpl

    自动生成HTML、CSS和JS模版代码

  10. Trailing Space

    去掉文件中每一行头部以及尾部的空格(包括TAB)

  11. ConvertToUTF8

    用于支持GBK和GB2312编码

  12. Emmet

    前端神奇，自动生成代码，使用 Emmet 生成 HTML 的语法详解

  13. Gulp

    自动化工具

  14. HTML/CSS/JS Prettify

    HTML/CSS/JS代码优化

  15. SFTP

    SFTP/FTP上传工具，无需额外的FTP上传工具

  16. SublimeCodeIntel

  17. SublimeCodeGit

  18. SublimeLinter

    代码检查

  19. JSHint

    JS代码检查
